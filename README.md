# node-rest-api
practicing creation of rest api with node.js

REST - REpresentational State Transfer

client sends request to server
server responds with text/html, Google geolocation API, 
JSON, XML, URLEncoded, FormData



create local server using expressJS
create endpoints
create HTTP methods on those endpoints
handle errors
body-parser feature

cors - cross origin resource sharing
cors error - security error if client and server are not on the same machine
add some headers to allow client to access the server
